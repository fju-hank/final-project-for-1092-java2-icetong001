## 專題用途
此專題有關一個酒類販賣機，在疫情下，不能現場付錢，因此下單後直接連接到網路銀行轉帳。

## 使用方式
執行程式為 com.fju.Tester
輸入0 顯示清單(品種 價錢 濃度)
輸入1 顯示GUI
    GUI內可輸入欲購買的數量
        輸入數量後可確認訂單=>連結到網路銀行轉帳
輸入2 詢問是否18歲
輸入3 顯示新竹酒商資料
